# EEG-signal-processing-tutorial-using-MNE-
This notebook is a comprehensive collection of signal processing examples, primarily focused on neurophysiological data (EEG/MEG). It covers:

Synthetic Signal Generation: Demonstrations of generating white noise, non-stationary signals with varying frequencies, and time-domain transients from the frequency domain.
EEG/MEG Signal Processing with MNE: A detailed workflow for loading, preprocessing (filtering, epoching, channel selection, artifact rejection), and analyzing sample MNE data. This includes Time-Frequency Representation (TFR) using Morlet wavelets and various visualizations of Event-Related Potentials (ERPs) such as joint plots, topographical maps of peak latencies, and analysis of peak latency across channels.
Convolution Examples: Practical illustrations of applying different types of kernels (symmetrical and asymmetrical) to an EEG snippet using convolution.
